# Model Keypoint YOLOv7

Deploy the keypoint [YOLOv7](https://github.com/WongKinYiu/yolov7) model on open-source [VDP](https://github.com/instill-ai/vdp) for keypoint detection.

This repo contains the ONNX graph of the pre-trained keypoint [YOLOv7](https://github.com/WongKinYiu/yolov7/releases/download/v0.1/yolov7-w6-pose.pt) model.
